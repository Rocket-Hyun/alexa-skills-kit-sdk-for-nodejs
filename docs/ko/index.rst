*******************
ASK SDK for Node.js
*******************

The ASK SDK v2 for Node.js는  boilerplate code를 작성하는 시간을 줄이고 기능 추가에 더 많은 시간을 쏟을 수 있게 함으로써 인기 있는 스킬을 개발하기 더 쉽게 만들어 줍니다.

SDK를 사용하여 개발을 시작하려면 아래 리소스를 참조하세요.
